# DrawVault

**Play. Win. Give Back.**

DrawVault is a motion-driven subscription platform that integrates performance tracking, a monthly lottery-style draw system, and a charity contribution model. Players compete in monthly draws, win real prizes based on their performance, and automatically contribute to the charities they care about.

## 🌟 Features

- **Monthly Performance Draws:** Log up to 5 performance scores throughout the month. Better play improves your odds in the algorithmic draw.
- **Charity Integration:** A dedicated portion (30%) of the subscription revenue goes directly to user-selected charities.
- **Prize Pools:** 50% of revenue forms the prize pool, structured across multiple tiers for fair and exciting rewards.
- **Premium Design:** Built with a modern, glassmorphic UI, rich typography, and smooth micro-animations powered by GSAP.
- **Transparency:** Results and funds distribution are completely transparent and verifiable.

## 🚀 Tech Stack

- **Frontend:** Vanilla HTML5, CSS3, JavaScript
- **Animations:** GSAP (GreenSock Animation Platform) & ScrollTrigger
- **Styling:** Custom CSS Variables, Glassmorphism, CSS Grid & Flexbox
- **Typography:** DM Serif Display, Outfit (Google Fonts)

## 📁 Pages & Structure

- `index.html` - The main landing page detailing how the platform works and its impact.
- `subscription.html` - Subscription configuration, plan selection, and charity allocation.
- `dashboard.html` - The user's main hub for tracking stats, upcoming draws, and active charities.
- `scores-draw.html` - Where players log their monthly performance scores.
- `results-verify.html` - Transparent verification page for monthly draw outcomes.
- `charity.html` - Detailed view of charity impact and supported organizations.
- `admin.html` - Platform control center for managing operations.

## 🛠️ Running Locally

This project is built completely static. You don't need any complex build tools to run it!

1. Clone or download the repository.
2. Open the directory containing the files.
3. Simply launch `index.html` in your favorite modern web browser, or use a local development server like VS Code's "Live Server" extension for the best experience.
